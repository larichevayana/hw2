# HW2 (Система для управления рецептами)

Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.

## Использование

Для запуска тестов выполните pytest, для использования кода импортируйте классы из модуля recipes.py.
